# Flask アプリ　専用テンプレート
nao708810 作業用

## How to use

1. アプリケーションをローカルにクローンします。

```cmd
git clone https://github.com/nao708810/flask-template.git
```

2. appsフォルダに移動します。

```cmd
cd apps
```

3. パッケージをインストールします。

```cmd
pip install -r requirements.txt
```

4. apps\config.sampleファイルをコピーして、config.pyにリネームします

5. アプリケーションを起動します。

```cmd
python manage.py
```
